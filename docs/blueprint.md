# **App Name**: Kerala PSC Genius

## Core Features:

- AI Question Generation: Generate multiple-choice questions (MCQs) based on the Kerala PSC syllabus, using an AI tool.
- Question Display: Display the MCQ with options in a clear and user-friendly interface.
- Answer Submission: Allow users to submit their answers for each question.
- Answer Evaluation: Evaluate the submitted answer and provide immediate feedback (correct/incorrect).
- Descriptive Explanation: Show a detailed explanation of the correct answer, regardless of whether the user answered correctly or not.
- Question Saving: Save questions, answers, and explanations in Firestore for future revision.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and knowledge.
- Background color: Light gray (#F5F5F5) for a clean and distraction-free learning environment.
- Accent color: Orange (#FF9800) to highlight important elements like correct answers or explanations.
- Headline font: 'PT Sans' (sans-serif) for question titles and section headers.
- Body font: 'PT Sans' (sans-serif) for question text, options, and explanations.
- Use clear and simple icons for navigation and feedback (e.g., checkmark for correct, cross for incorrect).
- Clean and organized layout with clear sections for question, options, and explanation.
